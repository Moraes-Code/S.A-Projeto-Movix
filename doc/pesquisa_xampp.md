## PESQUISA  SOBRE O XAMPP                                

### O que é Xampp?

    O XAMPP é um pacote com os principais servidores de código aberto do mercado, incluindo FTP, banco de dados MySQL e Apache com suporte às linguagens PHP e Perl. Com ele, é possível rodar sistemas como WordPress e Drupal localmente, o que facilita e agiliza o desenvolvimento. 

### Principais componentes do ambiente: 

   Apache, MySQL/MariaDB, PHP, phpMyAdmin.

### Finalidade de cada componente: 

   Apache: servidor web responsável por abrir e exibir os projetos no navegador através do localhost.

   MySQL/MariaDB: banco de dados utilizado para armazenar e organizar informações das aplicações.

   PHP: linguagem de programação usada para criar páginas dinâmicas e sistemas web.

   phpMyAdmin: ferramenta visual que permite administrar bancos de dados de forma simples e prática.

### Como realizar a instalação e configuração básica: 

    Acesse o Site Oficial do XAMPP  ,baixe o instalador no site oficial e Selecione os componentes que deseja instalar. É recomendado deixar todos marcados (Apache, MySQL, PHP, phpMyAdmin, etc.) e execute, mantenha o diretório padrão e avance. Abra o painel de controle e clique em "Start" ao lado do Apache e do MySCL para iniciar o seu servidor local. 
      Você pode gerenciar seu banco de dados clicando no botão Admin ao lado do MySQL no painel de controle ou digitando http://localhost/phpmyadmin diretamente no seu navegador. 




### Importância do ambiente para desenvolvimento local:


   O XAMPP é importante porque permite criar, testar e executar sites e sistemas diretamente no computador, sem precisar de hospedagem na internet. Ele facilita o desenvolvimento de aplicações PHP ao reunir ferramentas como Apache, MariaDB e PHP em um único ambiente, tornando os testes mais rápidos, seguros e práticos para os desenvolvedores.
