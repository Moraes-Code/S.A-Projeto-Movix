# S.A Projeto Movix

## Descrição
  O Movix é um sistema ferroviário desenvolvido para gerenciar informações de usuários e operações relacionadas ao controle de dados do sistema.

## Objetivo
  Desenvolver uma aplicação web organizada e funcional, aplicando conceitos de programação, banco de dados e interface gráfica.

## Funcionalidades
- Sistema de login
- Cadastro de usuários
- Gerenciamento de dados (CRUD futuramente)
